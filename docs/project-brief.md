# Project Brief: 生成式黄历小票 / Receipt Almanac

## 项目方向

这是一个全新的独立 MVP，用来开发“生成式黄历小票 / Receipt Almanac”。

产品目标不是 A4 海报，也不是复杂平台，而是：

- 用户输入今日状态、计划或一句描述；
- AI 输出结构化 JSON；
- 前端把文字完整渲染成一张窄长 receipt 风格小票；
- 用户可以保存图片，作为演示闭环。

## MVP 范围

第一版只覆盖以下流程：

1. 输入今日状态。
2. 生成结构化黄历内容。
3. 前端渲染 receipt 小票。
4. 导出 PNG 图片。

## 关键约束

- 前端完全重新搭建，不复用旧项目的前端组件、页面、CSS、海报模块。
- 所有文字由前端渲染。
- 小票视觉由前端实现。
- MVP 不做 AI 图片生成。
- 后续可以扩展 AI 拼贴素材，但不属于当前版本。

## 视觉方向

- 像真实 receipt / thermal paper / ticket stub
- 米白纸张、热敏打印、细密排版
- 不做星空、玄学、水墨、国风装饰
- 不做儿童化和糖果色

## AI 输出目标

AI 最终输出目标是 `ReceiptAlmanac` JSON，包括：

- 标题与副标题
- 票据编号
- 日期与农历信息
- 今日判断
- 宜 / 忌
- 辅助 meta 信息
- printedAt 与 barcodeValue
